AndTools
========

Android performance testing tools

Step:
1、点击“程序管理”，选择被测试程序；
2、点击“测试类型”，选择测试类型；
3、输入测试参数，保存；
4、点击开始测试；
5、历史记录可查看已测试的记录；
6、长按一条记录，可以删除该记录。

Description：
1、测试完成后会有系统通知，点击阅读通知可跳转到历史记录；
2、测试过程中程序尽量在后台运行，直到测试完成；
3、任何中断运行，测试结果都无法保存。
